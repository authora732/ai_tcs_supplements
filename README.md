# Anonymous Supplement: AI-Assisted Undergraduate Work on TCS Problems

This repository is an anonymized supplementary package for the paper **"What Should Future TCS Students Learn in the AI Era?"** It is designed to provide material that could not fit in the six-page SIGCSE-style paper while avoiding direct duplication of the paper text.

## What this supplement contains

- Extended case notes for five TCS-style problems.
- Prompt-engineering patterns used during the study, redacted and generalized.
- Verification records separating proved, computed, empirical, rejected, and future-work claims.
- Additional results and verification details that are not stated in the short paper.
- Extended literature-survey notes that support the journal version but are too long for the short paper.
- Templates for anonymized prompt logs.

## What this supplement intentionally does not contain

- No raw chat transcripts.
- No screenshots.
- No user names, university names, country names, email addresses, GitHub handles, or supervisor names.
- No claim that AI solved the original open problems fully.
- No direct copy of the six-page paper.

## How to read this repository

A suggested reading path is:

1. `prompt_engineering/prompt_protocol.md`
2. `additional_results/five_additional_results_not_in_short_paper.md`
3. `verification_records/claim_status_table.md`
4. the five files in `problems/`
5. `literature_survey/extended_literature_survey.md`

## Anonymity note

Before linking this repository during anonymous review, mirror it through an anonymizing repository service or create a fresh anonymous repository. Do not expose commit history from a personal account.
