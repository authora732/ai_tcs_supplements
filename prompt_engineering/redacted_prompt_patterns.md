# Redacted Prompt Patterns

These are generalized examples that can be included in a supplementary repository without revealing raw chats.

| Stage | Redacted prompt pattern | Why it mattered |
|---|---|---|
| Definition | "Explain this from zero; give examples before proof." | Prevented premature proof attempts. |
| Search | "Try constructions other than the obvious baseline." | Produced non-obvious candidates. |
| Exact check | "Compute the exact value and show the enumeration logic." | Made outputs verifiable. |
| Refutation | "Try to disprove your own lemma." | Exposed false proof steps. |
| Literature | "Do not assume this is open; search older names." | Found solved-status case. |
| Scope | "What exactly is proved, and what is only empirical?" | Prevented overclaiming. |
| Motivation | "This cannot be your best; try another direction." | Increased breadth but still required checking. |
| Audit | "List possible bugs or hidden assumptions." | Found field-arithmetic and theorem-scope issues. |
