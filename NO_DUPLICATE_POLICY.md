# No-Duplicate Policy

This supplement is not a second copy of the six-page paper. It follows these rules:

1. It may name the same five cases because it is a supplement.
2. It does not repeat the abstract, introduction, or discussion paragraphs from the paper.
3. It adds details that are too long for the paper: prompt patterns, claim-status labels, rejected claims, and literature-survey notes.
4. When a detail overlaps conceptually with the paper, the supplement gives extra verification context rather than reusing the same prose.
5. Raw private logs are excluded.

A small amount of overlap in terminology is unavoidable: mathematical objects such as $C_5[K_2]$, Barber's theorem, Borsuk, $K_{4,4}$, and Graphic TSP must be named for the supplement to be meaningful.
