# Anonymity Checklist

Before sharing this repository with reviewers, check the following.

- [ ] Repository owner name is anonymous.
- [ ] Commit author names and emails are removed or anonymized.
- [ ] No screenshots are included.
- [ ] No raw ChatGPT or Claude exports are included.
- [ ] No institution, country, professor, or student names appear.
- [ ] File metadata does not contain personal names.
- [ ] Repository URL does not contain a personal handle.
- [ ] The paper refers only to an "anonymized supplementary repository."
- [ ] The supplement uses role labels such as `students`, `expert supervisor`, and `AI system`.
- [ ] All mathematical claims are labeled by status: proved, computed, empirical, rejected, or future work.
